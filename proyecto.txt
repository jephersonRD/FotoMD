Proyecto: Plus Editor / Photo Plus

Desarrolla una aplicación web moderna llamada “Plus Editor” (nombre alternativo: “Photo Plus”) enfocada en la edición y optimización de imágenes desde dispositivos móviles Android y ordenadores.

Objetivo principal

Crear una plataforma web rápida, profesional y completamente responsiva que permita a los usuarios subir varias imágenes al mismo tiempo y aplicar herramientas de edición avanzadas de manera sencilla.

La aplicación debe estar desarrollada únicamente con:

- HTML5.
- CSS3.
- JavaScript moderno (ES6+).
- Debe funcionar como una PWA (Progressive Web App) instalable en Android.
- Código limpio, modular y bien comentado.
- Diseño optimizado para pantallas móviles y escritorio.

Paleta de colores

Usar exclusivamente:

- Negro profundo (#0D0D0D).
- Blanco (#FFFFFF).

Estilo visual

- Interfaz premium y moderna.

- Iconos profesionales.

Compatibilidad

Optimizar para:

- Navegadores modernos.

---

Pantalla principal

Al abrir la aplicación, mostrar:

- Logo de Plus Editor.
- Botón principal: «Subir imágenes».
- Contador para elegir cuántas imágenes se subirán.
- Zona de arrastrar y soltar archivos.
- Vista previa en cuadrícula.
- Indicador del tamaño de cada imagen.
- Barra de progreso durante el procesamiento.

---

Flujo de usuario

Paso 1

El usuario indica cuántas imágenes desea subir.

Ejemplos:

- 1 imagen.
- 10 imágenes.
- 20 imágenes.
- Sin límite estricto.

Paso 2

El usuario selecciona las imágenes desde:

- Galería.
- Archivos.
- Cámara del teléfono.

Paso 3

Una vez cargadas las imágenes, abrir automáticamente un menú con la pregunta:

«¿Qué quieres hacer con estas imágenes?»
